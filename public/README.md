# Public Assets Folder

Place your static assets here:

- **Logo**: `logo.png` or `logo.svg` (full logo with text)
- **Icon**: `icon.png` or `icon.svg` (icon only, square format)
- **Favicon**: `favicon.ico` (traditional favicon)
- **Apple Touch Icon**: `apple-touch-icon.png` (180x180px)
- **Other images**: Any other static images used across the site

Files in this folder are accessible from the root URL:
- `/logo.png` → `public/logo.png`
- `/favicon.ico` → `public/favicon.ico`
